Quick install guide

Select one device

Only one instruction set opens at a time, so customers do not have to sort through steps that do not apply to them.

FS

Device instructions

Firestick

  1. Install the Downloader app.
  2. Open Firestick settings and turn on developer access using one of these paths:
    • Device and Software -> About -> My TV, then keep clicking until developer mode is enabled.
    • My Fire TV -> About -> Fire TV Stick 4K, then keep clicking until developer mode is enabled.
  3. Go back one menu.
  4. Turn USB Debugging on.
  5. Turn Developer Options on.
  6. Turn Install From Unknown Apps on for Downloader.
  7. Open Downloader and enter 8965600 for Flix Pro.
  8. Optional: enter 396589 for Volume Boost, then download and install.
Final step: Open the app, go to Playlist or Account, and send a clear screenshot.
