# Flix-Pro Setup Website

Static website for Flix-Pro device setup instructions. It is ready for free hosting on GitHub Pages.

## Files

- `index.html` - main website
- `styles.css` - visual design
- `script.js` - device tab and QR card behavior
- `cards.html` - printable QR business cards
- `assets/flix-pro-hero.png` - generated hero image

## GitHub Pages Setup

1. Create a GitHub repository, for example `flix-pro-setup`.
2. Upload all files in this folder to the repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`, then save.
6. Your free website will be available at a URL like:

   `https://your-user-name.github.io/flix-pro-setup/`

## QR Code Links

Use these links for QR codes after replacing the domain with your real GitHub Pages URL:

- Firestick: `https://your-user-name.github.io/flix-pro-setup/#firestick`
- Android Phone: `https://your-user-name.github.io/flix-pro-setup/#android-phone`
- IPhone / IPAD: `https://your-user-name.github.io/flix-pro-setup/#ios`
- Android/Smart TV: `https://your-user-name.github.io/flix-pro-setup/#android-tv`
- Roku TV: `https://your-user-name.github.io/flix-pro-setup/#roku-tv`

Open `cards.html`, paste your real website URL, and print the generated business cards.
